Examples

Incremental timer

0
<ReactTimer
  start={0}
  end={value => value === 30}
  onEnd={value => console.log('ENDED WITH VALUE', value)}
  onTick={value => value + 1}
>
  {time => <div>{time}</div>}
</ReactTimer>

Decremental timer

30
<ReactTimer
  start={30}
  end={value => value < 25}
  onEnd={value => console.log('ENDED WITH VALUE', value)}
  onTick={value => value - 1}
>
  {time => <div>{time}</div>}
</ReactTimer>

Infinite timer

0
<ReactTimer
  start={0}
  end={() => false}
  onEnd={value => console.log('ENDED WITH VALUE', value)}
  onTick={value => value + 1}
>
  {time => <div>{time}</div>}
</ReactTimer>

Basic Clock

Mon Sep 07 2020 02:05:06 GMT+0000 (Coordinated Universal Time)
<ReactTimer
  start={0}
  end={() => false}
  onEnd={value => console.log('ENDED WITH VALUE', value)}
  onTick={value => value + 1}
>
  {() => new Date().toString()}
</ReactTimer>
